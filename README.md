## Pytest built-in Command Line Parameters/Options for project :

* --html

* --junitxml

    usage  : Generate junit report for test result
    
    option : Path-for-report-file
    
## Commmand to run script :

   py.test -v -s -m "marker" --html=<path> --junitxml=<path>
